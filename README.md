## 🚀 Hướng dẫn sử dụng

### Deploy lên GitHub Pages

1. Fork repo này
2. Vào **Settings** → **Pages**
3. Chọn **main branch** → **Save**
4. Truy cập `https://fitallest.github.io/portfolio/`

### Chạy local

```bash
git clone https://github.com/fitallest/portfolio.git
cd portfolio-website
open index.html
```
